# 行业周期与竞争格局分析参考

## 触发场景

当用户提问包含以下意图时，使用本参考：

- 市场份额有没有在扩张
- 公司在行业里竞争力强不强
- 行业景气度如何 / 处于周期哪个位置
- 公司是行业α还是跟随行业β
- 这个行业竞争格局在恶化吗
- 龙头优势在加强还是减弱

---

## 核心规则组合

| 规则 | 作用 |
|------|------|
| `market_share_change` | 判断公司是在跑赢行业（α）还是跑输行业（β） |
| `gross_margin_trend` | 判断竞争格局是否在恶化（毛利收窄往往是价格战或成本压力的前兆） |
| `roe_level` | 判断公司是否具备长期竞争优势（持续高ROE = 护城河） |
| `profit_leverage` | 判断行业景气上行时公司有没有释放经营杠杆 |

---

## 行业周期四象限判断

将 `market_share_change` 与 `gross_margin_trend` 组合，识别公司所处位置：

| | 毛利率扩张 | 毛利率稳定/收缩 |
|---|---|---|
| **市场份额扩张** | 🟢 景气上行 + 龙头化：最优信号，公司量价齐升，具备定价权和规模优势 | 🟡 以价换量：份额在扩但毛利在收窄，可能是通过降价或投入抢市场，需关注利润释放时间点 |
| **市场份额稳定/萎缩** | 🟡 行业涨价潮：行业整体受益于供给收缩或成本下降，非公司个体竞争力，需判断持续性 | 🔴 竞争恶化或需求萎缩：份额和毛利双收缩，公司可能处于竞争失势或行业下行周期 |

---

## 竞争优势判断

### 护城河信号（ROE 持续高于 15%）

`roe_level` = excellent 且持续多期：
- 说明公司具备某种可持续的竞争优势（品牌、规模、网络效应、低成本等）
- 结合 `market_share_change` = gaining，表明优势仍在强化
- 结合 `gross_margin_trend` = stable 或 expanding，表明定价权稳固

### 竞争优势弱化信号

- `roe_level` 从 excellent 降至 good 或 weak，持续 2 期以上
- `market_share_change` = losing + `gross_margin_trend` = contracting
- → 公司可能正在失去竞争地位，需检查是管理问题、产品老化还是颠覆性竞争

---

## 经营杠杆释放判断

景气上行期，龙头公司应能释放经营杠杆（固定成本摊薄，利润弹性大于收入弹性）：

- `profit_leverage` = leveraged（利润增速 > 收入增速 10pp）+ `market_share_change` = gaining
  → 公司正处于量增 + 利润释放的黄金阶段

- `profit_leverage` = diluted（利润增速 < 收入增速 -10pp）+ 收入增长
  → 增收不增利，可能是扩张期费用前置投入（短期可接受）或竞争导致的利润侵蚀（需警惕）

---

## 解释原则

- 区分**公司个体信号**（α）和**行业整体趋势**（β）
- `market_share_change` 衡量的是相对行业的表现，必须有可靠的行业增速数据才有意义
- 若行业增速数据缺失，`gross_margin_trend` 和 `roe_level` 可作为竞争格局的间接观测量
- 单季度数据波动较大，建议使用滚动4季度或年度数据做趋势判断

---

## 行业差异注意事项

- **强周期行业**（钢铁、煤炭、化工）：市场份额变化受价格周期驱动，需区分量与价的贡献
- **互联网 / 平台**：份额扩张期可能主动亏损（利润杠杆为负），需结合商业模式判断
- **消费品**：毛利率是核心竞争力指标，`gross_margin_trend` 权重最高
- **制造业**：`asset_turnover` 和 `capex_intensity` 应与竞争分析联动，效率领先是竞争优势的体现
